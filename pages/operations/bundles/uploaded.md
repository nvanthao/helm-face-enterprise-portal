# Uploaded Support Bundles

View and manage all uploaded support bundles from both Linux (Embedded Cluster) and Helm installations. Bundles are stored in the Replicated vendor portal and can be analyzed by the support team.

## Bundle Upload History

The upload history shows all bundles previously uploaded from HelmFace, including:
- Bundle ID and slug (for reference)
- Upload timestamp
- Source (Helm or Linux/Embedded Cluster)
- Status (successful upload, failed, etc.)

Bundles can also be uploaded directly from the HelmFace UI support page (`/support`) via drag-drop or file picker.
